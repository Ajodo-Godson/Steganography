# Steganography
A tool that encrypts a secret file (like a PDF or text document) and mathematically embeds it deep inside the frequency domain of a JPEG image or WAV audio file, making it completely undetectable to the human eye/ear and resistant to compression.

Update: I'm yet to fully implement this.... but I'm working on extending this to deal with larger file sizes...
The bottleneck is with the enryption and (of course) dealing with differnt formats.
